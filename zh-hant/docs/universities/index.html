            大學選擇 #

            本科後申請碩士還是博士 #

            很多國家都可以本科畢業後直接申請博士，部分國家的博士是帶薪的，部分國家的博士雖然要繳錢但也提供許多研究助理（RA）和助教（TA）的賺錢機會。

            而我們看到許多人的選擇是先讀碩士學位再申請博士，其原因主要有兩點：

            1. 本科科研經歷不夠；
            2. 該學校偏保守，即非自己學校畢業的學生會有一定的歧視。

            但如果您有科研經歷、成績也很好、同時也能提供行業中有話語權人物的推薦信，那麽完全可以申請博士試一試。畢竟碩士的學費如果沒有獎學金或非全額獎學金的話，還是比較苦惱的。

            此外，如果您希望找到工作，而不是在學術界繼續生涯，碩士也能提供更靈活的選擇，因為其只有一到兩年。

            但無論是申請碩士還是博士，都請對自己申請的項目、研究的領域做足夠的了解，然後再定決心。

            獎學金 #

            LGBTQA+群體的獎學金 #

            許多機構會提供受助者為LGBTQA+的獎學金，此處我們列舉幾個項目：

            • Grad School Application Funding - ostem.org
            • LGBT Scholarships Around the World - QS
            • Scholarships & Financial Aid For LGBTQ Students - gograd.org

            可以去廣泛的搜索一下，關鍵詞舉例：

            lgbt, queer, transgender, scholarship, [STEM, Science, Art, etc.]

            大學獎學金 #

            各大學會在入學前或入學後按成績、表現等為學生提供一定的獎學金。比如在美國詞條下，有評論者指出「即使是碩士，足夠獎學金能夠把成本拉到歐洲水準」。

            此部分的獎學金明細需要咨詢學校相關部門。

            研究方向獎學金 #

            部分基金會會向一些潛在的研究領域提前投入資金來資助研究，比如生物安全、人工智能安全、環境問題等。

            部分機構舉例：

            • Open Philanthropy
            • Center on Long-Term Risk

            從開始到申請（簡版） #

            如果希望獲得更詳細的資訊，請看本頁後半部分。

            在選擇大學中，最難的一步實際上是找到目前的成績和就讀學校匹配的目標大學範圍。因為我們沒有經驗告訴我們哪一個範圍是我們所匹配的。

            但除開這一步，我們還是能夠完成相當一部分工作的。

            第一步：選出國家範圍 #

            這一步主觀性比較強，你可以結合我們的國家和地區介紹來給自己劃定一個大致的範圍。但是這個範圍不一定是死的，甚至它可能要比你最後選擇的範圍要寬泛得多，這都沒有關係，你完全可以多畫一點範圍，從而增大你選擇的空間。

            第二步：學校列表 #

            首先我們需要準備一些工具，比如說各種的學校排名，這種榜單主要用來讓你了解自己可以選擇哪些學校，但請務必不要被排名所困，更重要的是自己活得開心、能在相對喜歡的領域做研究。比如： QS / THE / USNews / ARWU / CWUR 等等。排名並不能代表學校的實際水平，因為不同排名的側重點不一樣，部分側重點可能是我們並不在意的，但是對於我們如果沒有其它的信息來源的話，這個參考可能也是比較有價值的。同時注意它們往往每年都會進行排名，所以說可能您要注意去切換到最新的年份。您可能需要特別關注的幾個指標：「教學質量」（對於非研究階段的學習者）、「就業率」、「畢業薪資」等。

            還有一點需要提醒的是，學校排名不代表學科排名，其學科排名也是一個值得考量的因素，尤其是要做研究的話。也可以通過一些學科排名機構看到，雖然這些機構同樣不甚準確，但是它是免費的開放數據，有相當的參考價值。

            在這些排名當中篩選出你所喜歡的國家或幾個國家的高校，比如可以做一個列表。列表當中包含了，你所要去篩選的所有國家。

            第三步：項目列表 #

            然後，我們要去找這些學校中，你所喜歡的專業/項目。

            具體來說，我們要在Google或其它搜索引擎中搜索這個學校的名字，進入官網。找到招生計劃或者是項目介紹，然後找到它們所有的本科/碩士項目列表。這樣我們可以用篩選或者搜索的方式找到我們感興趣的專業。

            到這一步，你使用一個表格來記錄所有的你找到的項目的詳細信息。比如這個項目的語言要求、GPA要求、學分要求等。此外往往項目的頁面上還包含了學費和獎學金的信息，這個也是很重要的考量因素。

            這樣，我們把所有學校的可申請項目整理出來後，若您處在本科的前期或者高中階段你就可以按照項目要求繼續準備；如果將要到達到申請階段，那麽也可以開始排除一些沒有滿足條件的項目。

            第四步：正式申請 #

            到這一步，我們或許就沒有什麽特別的細節可以講述了，因為每個學校的要求都不一樣，也就是可能要求提交的材料和闡述的內容也不一樣。

            祝大家申請順利。

            從開始到申請 #

            選擇國外大學項目和篩選適合自己的項目是一個重要的決策過程。總體來說，信息比國內填報志願更透明，但搜集整理信息的過程也相對來說會更繁瑣。

            本文包含了對於未到申請階段的同學的一些建議，比如如何選擇，以及對於已經進入申請階段的同學的一些建議。本文以碩士申請為例。

            選擇國家和地區 #

            國家部分可以參考本手冊的「國家與地區」部分，此處不贅述。

            通常來說，國外學術機構更加強調包容性和平等多元，因此，一般情況下，學術機構可能比社會普遍情況更為包容。但需要注意的是，在一些學校中仍可能存在推諉和歧視問題。此外，也可能存在部分教授較為保守，針對性別多樣性群體可能存在不公平對待的情況，然而，因為各種原因，此類不公平對待可能很難明確定義為歧視，進而很難得到解決。因而，適當了解該學校內部的性多樣性群體的反應與學校相關政策的是有必要的。

            對於具體的學校範圍選擇，我們也大概可以總結出一定的規律。

            • 在同一個地域範圍內，通常在經濟較發達的城市，對性多樣性群體的包容度較高，這可能會使我們在校園以外的生活更加便利——比如購置生活用品等。
            • 在大城市學習也往往會提供更廣泛的就業前景，例如更多實習機會等。以瑞典查爾姆斯大學為例，由於其位於沃爾沃總部所在地，相當一部分畢業生可以直接在沃爾沃進行實習，從而獲得更好的就業保障。
            • 大城市的生活費用通常較高，尤其是在學校不提供宿舍或沒有學生住房機構的情況下。作為結果，生活費用的開銷會比較高，因而根據生活水平仔細衡量自己的預算也是重要的。

            自然，不同國家、不同州、甚至不同學校可能有不同的優惠政策。因此，具體的生活費用高低以及是否有學生補貼等問題可能需要您進一步了解和獲取更多資料。

            整理學校清單 #

            假設我們已經初步選定了地域範圍，我們可以根據之前提到的開放包容程度、生活成本、就業或實習前景來篩選具體的學校。一般來說，學校的錄取難度與其排名是正相關的，因而，我們在一定程度上可以根據排名來選擇可供選擇的學校列表。然而，也需要提及的是，在美國，相對來說一個 USNews 排名可以相對反應難度；然而歐洲的不同國家的難度也會不同，也就是即便排名相近，可能不同國家的國際生錄取比例和評價方式都會影響到難度，因而在選擇歐洲學校的時候，可能需要更謹慎地考量。

            假設，我們學校排名來篩選可選的院校列表，這個目標院校的範圍實際上需要綜合許多因素，比如 GPA、本科學校、課外活動、科研成果等。從某種意義上說，這個目標排名實際上需要仔細評估的，然而我無法提供具體參考，因為不同學校對學生的評價方法不同。同時，個人的情況也千差萬別，所以，筆者在這里無法提供一個簡單的方案。因而此處便跳過如何選擇目標院校範圍。但可以提出的是，如果有一定的科研經歷或論文發表，可以適當將自己的目標稍微提高一點。

            整理項目 #

            對於碩士學位來說，專業選擇可能相對寬松一些。如果是直博，基本上只能選擇與本科所研究的方向緊密相關的項目/導師，也就是「轉行」的可能較為有限。但如果選擇攻讀碩士學位，那麽可以在一定程度上更自由地調整自己的專業方向，以符合個人的興趣或就業需求。然而，同樣需要指出的是，許多學校仍將碩士項目與本科項目的匹配程度作為一個重要考量指標。因此，如何把握這種差異和風險也是我無法給出答案的。

            然後，我們來談談如何具體尋找適合自己申請的項目，這是非常重要的一步。

            首先，根據上面的方法，我們已經獲得了學校清單。下一步，我們就要訪問這些學校的官方網站——盡量選擇官方網站，因為許多學校可能有專門的招生網站，或者存在一些中國或其他地方有分校的話，就會有一些分站。甚至不排除有假冒官方網站的可能。官方網站通常使用.edu或.edu.xx等國家域名後綴，或者有些學校使用.ac域名，這樣的域名通常表明網站是正規的。在官方網站上，我們通常會發現在 header 部分有一個名為 Applying/Programs/Admissions 的按鈕/下拉菜單，這些詞匯往往表示我們可以從此處獲得更多的關於錄取或項目的信息。

            此外，大部分學校會對項目列表提供篩選，因而我們可以在篩選後，瀏覽感興趣的項目。然後，仍然是通過表格來管理我們收集到的所有信息。這個表格可以包含語言要求、語言成績、GRE、GMAT等考試要求，以及GPA要求、學分要求和數學學分要求等。同時在後面的行中標記自己尚未滿足的要求，特別是如果您目前還在大一到大三階段，因為這樣您可以有針對性地調整自己未來的選課計劃，以符合要求。此外，如果希望簡化後續的更新流程，還可以在列中加入項目簡介、項目網址等信息，這樣未來篩選可能會方便一點。

            學校專業TOEFLGREGPA其他要求未滿足（其他）
            A 大學X 專業1003404.5/5
            A 大學Y 專業903303.7/4
            B 大學XY 專業95330未明確要求

            如果這些項目都有學分要求，例如特定的學分數量，我們需要另外創建一個單獨的工作表來計算自己的學分。例如，你可以將自己的成績單變成表格——將每門課程的名稱對應的學分和獲得的績點記錄在一個工作表中，並放置在不同的列組中。例如，ABC 三列的是思政類課程，DEF 三列是將數學類課程，GHI 三列是計算機類課程等。以此類推。通過這樣的方式，我們可以使用公式功能核算出每個類別獲得的學分。

            此外，需要注意的是，不同地區的學分統計方法不同，因而可能需要進行轉換。假設，我確實記不住具體的數字了抱歉，中國的畢業要求是 240 個學分，而歐洲可能只需要 120 個學分。在這種情況下，中國的學分需要 * 0.5 來獲得核算後的學分。同樣，我們可以使用公式來計算每個類別核算後的學分轉換後的學分，並與前面的工作表中的所有要求進行對比，看是否滿足條件。

            最後，如果你仍然處於大二或大三階段，也就是尚未完成所有課程，你可以根據相應的差值來選擇課程。但如果你已經到了申請階段，例如大四的上學期，可能時間有點緊迫，那麽你可以根據這張表格刪除一些自己喜歡但未達到要求的學校，以節省申請費用。
