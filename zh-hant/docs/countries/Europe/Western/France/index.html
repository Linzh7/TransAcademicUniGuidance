            居留 #

            關於財產證明，法國在 France Visas 中對不同目的的申請有不同的要求，可以根據您的目的使用上述連結查看需準備文件。

            可供參考的標準是：

            • 每月至少相當於615歐元（即 7380 歐元/年）的資金證明；
            • 獎學金證明；
            • 或法國銀行賬戶存款證明；
            • 或資助人的資助證明，同時該資助人必須證明有足夠的、可信的、穩定的資金來源，並附其身份證件覆印件。

            在法國高效學習至少兩年後，畢業可直接申請加入法國國籍；或在法國居住五年以上亦可申請。
