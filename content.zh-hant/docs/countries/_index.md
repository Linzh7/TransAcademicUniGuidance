# 地區與國家

對於我們來說，選擇一個什麼樣的地區來學習、工作、生活是很重要的，因為在一個適合我們的地方，我們才能保證自己不受傷害、才能保證我們能更好的發展、才能保證我們有更多的力量去幫助其他人。
